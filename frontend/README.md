Frontend (Next.js) — Quick start

1) Install dependencies:

   cd frontend
   npm install

2) Start dev server:

   npm run dev

3) Open in browser: http://localhost:3000

Notes

- The frontend expects the backend API at `http://127.0.0.1:8000`.
- If your backend runs on a different host/port, update the fetch URL in `frontend/pages/index.js`.
